PRAE Refrigeración y Climatización Espinal - Frontend XAMPP

INSTRUCCIONES:
1. Copia la carpeta prae_refrigeracion_frontend dentro de htdocs de XAMPP.
2. Abre XAMPP y activa Apache.
3. Entra desde el navegador a:
   http://localhost/prae_refrigeracion_frontend/

ESTADO DEL PROYECTO:
- Versión frontend sin base de datos.
- Diseño actualizado tomando como referencia visual la página de Ferretería Super Jimmy, pero usando azul y blanco.
- Productos, servicios, ofertas, trabajos, consejos, ayuda, sobre nosotros y contacto están separados en páginas.
- La sección contacto queda reservada para completar datos cuando sean confirmados.
- El panel admin es visual/demo y se conectará al backend cuando se creen las tablas.

ESTRUCTURA:
- assets/css/main.css: diseño principal.
- assets/js/main.js: menú, carruseles, filtros, comentarios y WhatsApp.
- data/: datos temporales del frontend.
- includes/: header, footer y funciones.
- admin/: panel visual preparado.
- secciones/: carpetas reservadas por apartado.
